## Requirements

This course requires [mdBook](https://github.com/rust-lang/mdBook).
You can install it via:

```bash
cargo install mdbook
```

Or you can also install it via Homebrew:

```bash
brew install mdbook
```

## Build

To build this book use:

```bash
mdbook build
```

The compiled book is written to `./book`.
Open `./book/index.html` with your preferred browser to access it.
